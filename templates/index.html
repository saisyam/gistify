Gistify

Embed any source file from Github into your webpage with syntax highlighting

Embed any source file

Github supports only Gists to be embedded inside a webpage. With Gistify you can embed any source file from any public repository.

Same highlighting as Github

Gistify uses the same stylesheet used by Github for displaying its Gists. You don't see any difference in the syntax highlighting.

Display specific line numbers

Gistify can display specific line numbers in the source file. This will help in better explaining your work in your blog or webpage.

Embed any source file

Here I am embedding a simple source file from my repository, https://github.com/saisyam/gistify/blob/master/sample.py.

Simply append the Github URL to https://gistify.saisyam.com. For example, https://gistify.saisyam.com/https://github.com/saisyam/gistify/blob/master/sample.py

Display specific line numbers

To display specific line numbers, you have to append line numbers along with the source file URL. To display line numbers 1,2 and 7 to 9, you have to append /l/1,2,7-9.

Note: Don't forget to append "/l/"

For example, https://gistify.saisyam.com/https://github.com/saisyam/gistify/blob/master/sample.py/l/1,2,6-8

Embed Github contributions graph

I always love to project my contributions graph on my blog or website. Github does not provide the way to embed contributions as it does for gists. I added a feature to Gistify to display contributions graph. All you have to do is to append the Github username to the URL, https://gistify.saisyam.com/contribution

For example, https://gistify.saisyam.com/contribution/saisyam
